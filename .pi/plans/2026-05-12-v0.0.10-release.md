# v0.0.10 Release Plan (2026-05-12)

## Scope
- Publish a new release version from `0.0.9` to `0.0.10`.
- Commit current workspace changes.
- Push branch and `v0.0.10` tag to trigger GitHub release workflow.

## Steps
1. Bump workspace version in `Cargo.toml` to `0.0.10`.
2. Bump `install.sh` `FALLBACK_VERSION` to `0.0.10`.
3. Run release build verification for `dcode-ai-cli`.
4. Commit all changes with a release commit message.
5. Push `main` and push tag `v0.0.10`.

## Verification
- `cargo build --release -p dcode-ai-cli`
- `dcode-ai --version` from built artifact path

## Publish Path
- GitHub Actions `release.yml` runs on `v*` tags and creates release artifacts.
