# v0.0.13 Hotfix Release Plan (2026-05-13)

## Problem

`v0.0.12` was published before commit `8d87253` (startup approval popup
question routing fix), so installers pinned to `v0.0.12` still show:

- `question was no longer pending; cleared stale question state`

## Goal

Publish `v0.0.13` including the startup-popup routing fix.

## Plan

1. Bump workspace version:
   - `Cargo.toml` `workspace.package.version` -> `0.0.13`.
2. Update installer fallback:
   - `install.sh` `FALLBACK_VERSION` -> `0.0.12`.
3. Regenerate lock metadata + verify:
   - `cargo fmt --all -- --check`
   - `cargo test -p dcode-ai-cli`
4. Publish:
   - Commit release bump.
   - Push `main`.
   - Create and publish GitHub release tag `v0.0.13`.
